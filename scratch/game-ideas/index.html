Game Ideas for Grade 6 Teams

This page helps teams get unstuck fast. Pick a direction, build a first playable version, test it, and improve it.

Inspiration Areas

You can remix ideas across these styles:

starter30 minclickerTeam 2-3

Click Rush

Tap targets quickly before the timer runs out.

Core Loop: Spawn target -> click target -> gain points -> repeat faster.

First Build Goal: Start with one moving target, one score counter, and a 30-second timer.

Skills Needed: events, variables, timing

Starter Tutorial: Events + Timing

Stretch Ideas

  • Add combo points for fast clicks
  • Add two target types with different scores
  • Add start and end screens
builder60 minarcadeTeam 3-4

Paddle Duel

A two-player paddle game with first-to-5 scoring.

Core Loop: Move paddles -> bounce ball -> score when ball passes edge.

First Build Goal: Start with two paddles, one ball, one scoreboard, and first to 5 points.

Skills Needed: motion, variables, input

Starter Tutorial: Motion + Coordinates

Stretch Ideas

  • Add ball speed increase every 10 seconds
  • Add sound effects for paddle hits
  • Add pause and restart buttons
builder60 mincatchTeam 3-4

Catch and Sort

Catch falling items and sort good vs bad choices.

Core Loop: Move basket -> catch items -> update score/lives -> continue.

First Build Goal: Start with one basket and two types of falling items. Show score and lives.

Skills Needed: input, variables, clones

Starter Tutorial: Input Patterns

Stretch Ideas

  • Add level changes by speed
  • Add list of random item names
  • Add bonus item with rare spawn
builder60 minrunnerTeam 2-3

One-Button Jumper

Press one key to jump over obstacles and survive.

Core Loop: Jump -> avoid obstacle -> score distance -> repeat.

First Build Goal: Start with one jumper, one obstacle, and a game-over screen with restart.

Skills Needed: events, variables, broadcast

Starter Tutorial: Broadcasts + Scene Control

Stretch Ideas

  • Add double-jump power-up
  • Add day/night background cycle
  • Add high-score list
challenge90 minsimulationTeam 3-4

Pet Care Loop

Keep a virtual pet healthy by choosing smart actions.

Core Loop: Choose action -> change hunger/energy/happiness -> check status.

First Build Goal: Start with three pet meters (hunger, energy, happiness) and three care buttons.

Skills Needed: variables, input, looks

Starter Tutorial: Variables + State

Stretch Ideas

  • Add random events every 20 seconds
  • Add mini-game for bonus energy
  • Add pet evolution stages
challenge90 minquizTeam 4-5

Quiz Show Live

A classroom quiz with random questions and points.

Core Loop: Show question -> choose answer -> score -> next question.

First Build Goal: Start with five questions in a list, one score counter, and a win screen.

Skills Needed: lists, variables, input

Starter Tutorial: Lists + Collections

Stretch Ideas

  • Add a countdown timer
  • Add category chooser
  • Add team mode with two scores
builder60 minmazeTeam 2-3

Maze Rescue

Guide a character through a maze before time runs out.

Core Loop: Move through maze -> avoid walls -> reach goal.

First Build Goal: Start with one maze, one timer, and one finish spot.

Skills Needed: motion, sensing, debugging

Starter Tutorial: Debugging Workflow

Stretch Ideas

  • Add three maze levels
  • Add moving obstacles
  • Add best-time tracker
challenge90 minsurvivalTeam 3-4

Space Collector

Fly through space collecting fuel and avoiding debris.

Core Loop: Move ship -> collect fuel -> dodge hazards -> survive.

First Build Goal: Start with one ship, fuel and debris spawns, plus score and lives.

Skills Needed: input, clones, variables

Starter Tutorial: Clones + Spawners

Stretch Ideas

  • Add shield power-up
  • Add boss wave every 30 seconds
  • Add two ship choices
challenge90 minstoryTeam 4-5

Story Quest

Players choose actions and unlock different endings.

Core Loop: Read scene -> choose option -> jump to next scene.

First Build Goal: Start with three scenes and two possible endings.

Skills Needed: broadcast, variables, looks

Starter Tutorial: Looks + Scenes

Stretch Ideas

  • Add inventory choice checks
  • Add character mood meter
  • Add secret ending path
challenge90 minbuilderTeam 3-4

Mini Builder

Build and place objects on a simple map.

Core Loop: Pick tool -> place object -> earn build points.

First Build Goal: Start with two objects to place and one build score.

Skills Needed: my blocks, input, variables

Starter Tutorial: My Blocks (Functions)

Stretch Ideas

  • Add object rotation
  • Add map themes
  • Add save/load using lists

Pitch Your Idea

Before coding, every team writes these three answers in one minute each.

  1. What does the player do every 5-10 seconds in your game?
  2. What is the first version you can finish quickly?
  3. What one stretch feature will you add only after that version works?
