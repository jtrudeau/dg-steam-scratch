Lists + Collections

listsdata

Store sets of data like quiz questions, inventory, or high scores.

Milestones

  1. Collect: Use lists to store multiple items.
  2. Pick: Choose a random or ordered item from a list.
  3. Update: Add or delete items as the project changes.

Guide

Goal

Use lists for collections that grow over time.

Try it

Create a list of questions and display a random one.

when green flag clicked
add [Question A] to [questions v]
add [Question B] to [questions v]
say (item (pick random (1) to (length of [questions v])) of [questions v])

Checkpoint

You can store at least 3 items and pick one reliably.
