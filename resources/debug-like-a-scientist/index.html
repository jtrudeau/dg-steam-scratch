Debug Like a Scientist

debuggingeventslogic

A quick protocol for hunting bugs: observe, predict, isolate, and retest with intention.

Steps

  1. Pause and observe: Click the green flag and list what you expected vs. what actually happened before touching any code.
  2. Isolate one script: Right-click → duplicate a sprite or disable one script at a time to see if the bug follows.
  3. Add print-style clues: Drop `say` blocks or temporary color changes inside loops so you know which blocks are running.
  4. Check events and resets: Make sure green flag scripts set starting positions/variables and broadcasts match exactly.
  5. Retest and log: After each tweak, retest and jot one sentence in your debugging log so you can roll back or teach peers.
