  - title: "Teach a greeting"
    description: "In the Code tab, drag `when green flag clicked` and snap a `say [Hello DG!] for (2) seconds` block underneath. Your sprite now talks when the flag starts the story."
  - title: "Add a wiggle"
    description: "Slide in `move (10) steps` and `move (-10) steps` blocks to create a tiny dance. You can add a `wait (0.5) seconds` in between for rhythm."
  - title: "Show it off"
    description: "Click the green flag and watch your sprite wave hello. Ready for more? Add a key press block so it tells a joke when you press space!"
