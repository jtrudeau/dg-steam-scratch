## Goal

Develop a calm, systematic way to debug Scratch projects.

## Try it

Add temporary say blocks to confirm which script is running.

```scratch
when green flag clicked
say [Debug: script started] for (1) seconds
```

## Checkpoint

You can explain the bug in one sentence before fixing it.
