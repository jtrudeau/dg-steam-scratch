## Goal

Store and update values that matter to your project.

## Try it

Build a score counter that resets when the flag is clicked.

```scratch
when green flag clicked
set [score v] to (0)
change [score v] by (1)
```

## Checkpoint

Your variables reset reliably when the project restarts.
